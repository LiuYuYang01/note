# Python 列表操作

## 基本列表操作

**打印该列表中的每一项**

```python
l = ['apple', 'banana', 'cherry']

for i in l:
    print(i)
    # apple
    # ...
```



**打印列表中的每一项 含索引**

```python
l = ['apple', 'banana', 'cherry']

for i, d in enumerate(l):
    print(i, d)
    # 0 apple
    # ...
```

指定下标从 `10` 开始 `enumerate(l, 10)`



**打印 5 个数据 (0~4)**

```python
for l in range(5):
    print(l)
    # 1
    # 2
    # ...
```



**判断列表中某个数据是否存在**

```python
l = ['apple', 'banana', 'cherry']

print('banana' in l)  # True
print('strawberry' not in l)  # True
```



**合并两个列表 `+`**

```python
a = ['apple', 'banana']
b = ['cherry', 'strawberry']

print(a + b)
# ['apple', 'banana', 'cherry', 'strawberry']
```



**复制列表 `*`**

```python
l = ['apple']
print(l * 3)
# ['apple', 'apple', 'apple']
```



**比较列表的值是否相等**

```python
a = ['apple', 'banana']
b = ['apple', 'banana']

# 比较内容
print(a == b)  # True
# 比较地址
print(a is b)  # False
```



## 常用的列表方法

### append

将指定数据添加到列表中（默认添加到最后一位）

```python
l = [1, 2, 3]
l.append(4)
print(l)
# [1, 2, 3, 4]
```



### insert

在列表最前面添加一个数据：`666`

```python
l = [1, 2, 3]
l.insert(0, 666)
print(l)
# [666, 1, 2, 3]
```



在下标为 `1` 的位置添加数据：`666`

```python
l = [1, 2, 3]
l.insert(1, 666)
print(l)
# [1, 666, 2, 3]
```



### index

获取列表中数据：`b` 的下标

```python
l = ['a', 'b', 'c']
print(l.index('b'))
# 1
```



### count

获取列表中：`b` 元素出现的次数

```python
l = ['a', 'b', 'a', 'c', 'b', 'b', 'd']
print(l.count('b'))
# 3
```



### pop

从后往前删除列表中的数据

```python
l = ['a', 'b', 'c']

# pop没有参数则默认从后往前删除
list.pop()

print(l)
# ['a', 'b']
```



删除列表中指定下标的数据

```python
l = ['a', 'b', 'c']
l.pop(2)
print(l)
# ['a', 'b']
```



删除列表中倒数第二个数据

```python
l = ['a', 'b', 'c']
l.pop(-2)
print(l)
# ['a', 'c']
```



### del

根据下标删除列表中指定的数据

```python
l = ['a', 'b', 'c']
del l[2]
print(l)
# ['a', 'b']
```



### remove

删除列表中指定的数据

```python
l = ['a', 'b', 'c']

# 删除：'b'
l.remove('b')
print(l)
# ['a', 'c']
```



删除字典数据

```python
l = [
    {"id": 1, "name": "zs"},
    {"id": 2, "name": "ls"},
    {"id": 3, "name": "ww"},
]

l.remove({"id": 2, "name": "ls"})

print(l)
# [{'id': 1, 'name': 'zs'x}, {'id': 3, 'name': 'ww'}]
```



### clear

清空列表中的数据

```
l = ['a', 'b', 'c']

print(l) # ['a', 'b', 'c']
l.clear()
print(l) # []
```



### reverse

反转列表中的数据

```python
l = ['o', 'l', 'l', 'e', 'h']
l.reverse()
print(l)
# ['h', 'e', 'l', 'l', 'o']
```



### copy

拷贝列表中的数据

```python
a = ['a', 'b', 'c']
b = a.copy()
print(a, b)
# ['a', 'b', 'c'] ['a', 'b', 'c']
```



### sort

对列表中的数据进行排序

```python
l = [1, 3, 2, 4, 6, 5]

l.sort() # 默认从小到大排序
print(l)
# [1, 2, 3, 4, 5, 6]

l.sort(reverse=True) # 从大到小
print(l)
# [6, 5, 4, 3, 2, 1]
```



对字典中的 `id` 从大到小进行排序

```python
l = [
    {"id": 1, "name": "zs"},
    {"id": 2, "name": "ls"},
    {"id": 3, "name": "ww"},
]

l.sort(key=lambda x: x['id'], reverse=True)
print(l)
# [{'id': 3, 'name': 'ww'}, {'id': 2, 'name': 'ls'}, {'id': 1, 'name': 'zs'}]
```



也可以使用 `sorted`，相比 `sort` 更加安全一些，因为他不会改变原数据，而是将排序后的数据返回

```python
sorted(l, key=lambda x: x['id'], reverse=True)
```
