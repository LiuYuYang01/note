课程简介

主要内容

  • flask框架
  • flask-restful扩展包
  • flask-sqlalchemy扩展包

学习目标

  • 能够使用flask框架实现web应用的基本功能
  • 能够使用flask-restful扩展包实现基于REST风格的接口
  • 能够使用flask-sqlalchemy扩展包完成数据库的常见操作

学习时长

3天
